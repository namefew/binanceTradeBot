# 快速开始指南 🚀

## 5分钟快速体验

### 第一步: 安装 (1分钟)

双击运行 `install.bat` (Windows) 或在终端执行:

```bash
pip install -r requirements.txt
```

### 第二步: 配置API (1分钟)

1. 复制配置文件:
   ```bash
   copy .env.example .env
   ```

2. **获取测试网API密钥** (免费,无需真实资金):
   - 访问: https://testnet.binance.vision/
   - 用GitHub账号登录
   - 生成API Key和Secret
   - 填入 `.env` 文件:
     ```env
     BINANCE_API_KEY=你的测试网API密钥
     BINANCE_API_SECRET=你的测试网密钥
     BINANCE_TESTNET=true
     ```

### 第三步: 启动Web界面 (30秒)

```bash
streamlit run app.py
```

浏览器会自动打开 http://localhost:8501

### 第四步: 运行第一次回测 (3分钟)

在Web界面中:

1. **选择策略**: 双均线交叉
2. **配置参数**:
   - 交易对: BTCUSDT
   - 时间周期: 1h  
   - 回测天数: 90
   - 初始资金: 10000
   - 短期均线: 10
   - 长期均线: 30
3. **点击"开始回测"**
4. **查看结果**:
   - 总收益率
   - 最大回撤
   - 夏普比率
   - 资金曲线图
   - 交易信号图

🎉 恭喜! 你已经完成了第一次策略回测!

## 核心概念

### 什么是回测?
回测 = 用历史数据测试策略表现

例如: 
- 如果过去90天使用这个策略,能赚多少钱?
- 最大会亏损多少?
- 胜率如何?

### 什么是实时交易?
程序自动监控市场,根据策略信号执行买卖操作。

### 常见指标说明

| 指标 | 含义 | 好坏判断 |
|------|------|----------|
| 总收益率 | 总共赚了/亏了多少 | 越高越好 |
| 最大回撤 | 从最高到最低跌多少 | 越低越好 |
| 夏普比率 | 风险调整后的收益 | >1良好, >2优秀 |
| 胜率 | 盈利交易占比 | >50%较好 |

## 内置策略介绍

### 1️⃣ 双均线交叉策略
**原理**: 快线上穿慢线买入,下穿卖出

**适用行情**: 趋势明显的市场

**参数**:
- 短期均线(快线): 默认10
- 长期均线(慢线): 默认30

**示例**:
- 10日均线上穿30日均线 → 买入
- 10日均线下穿30日均线 → 卖出

---

### 2️⃣ RSI超买超卖策略
**原理**: RSI低时买入(超卖),高时卖出(超买)

**适用行情**: 震荡市场

**参数**:
- RSI周期: 默认14
- 超卖线: 默认30 (低于此值买入)
- 超买线: 默认70 (高于此值卖出)

**示例**:
- RSI从25上升到31 → 买入
- RSI从75下降到69 → 卖出

---

### 3️⃣ 布林带策略
**原理**: 价格触及下轨买入,触及上轨卖出

**适用行情**: 震荡市场

**参数**:
- 周期: 默认20
- 标准差倍数: 默认2.0

**示例**:
- 价格突破下轨向上 → 买入
- 价格突破上轨向下 → 卖出

## 常见问题速查

### ❓ 没有Python环境?
下载安装 Python 3.8+: https://www.python.org/downloads/

### ❓ pip安装很慢?
使用国内镜像:
```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### ❓ Streamlit打不开?
手动访问: http://localhost:8501

### ❓ API连接失败?
- 检查 `.env` 文件是否正确配置
- 确认使用的是测试网还是实盘
- 检查网络连接

### ❓ 回测没有交易?
可能原因:
- 策略参数不适合当前市场
- 数据量太少
- 尝试调整策略参数

### ❓ 如何停止程序?
按 `Ctrl + C`
